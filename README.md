# TesloShop

1. Clonar repositorio
2. clonar el archivo `.env.template` y renombrarlo a `.env`
3. Ejecutar comando `npm install` para instalar las dependencias 
4. Verificar que el BE esté corriendo en el puerto `3000` (https://github.com/AnaflaviaDiaz/be-nest-teslo-shop)
5. Ejecutar el comando `npm run dev` para iniciar el servidor de desarrollo
